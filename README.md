# 📊 Sales Dashboard

## Overview
This is an interactive E-Commerce Sales Dashboard built using Power BI. It helps analyze sales performance through interactive visualizations and key performance indicators (KPIs).

## Features
- 📈 Sales Performance Analysis
- 🌍 Region-wise Sales Analysis
- 📅 Monthly Sales Trends
- 📊 Category-wise Sales
- 🎯 Interactive Filters and Slicers

## Tools Used
- Power BI
- Microsoft Excel

## 📈 Key Insights

- 💻 Technology is the highest-selling category, generating approximately ₹103M in sales.
- 🪑 Furniture ranks second with around ₹71M in sales, while Office Supplies contributes the least (₹22M).
- 📱 Phones, Copiers, Accessories, and Machines are the most profitable sub-categories.
- 📚 Tables, Bookcases, Chairs, and Furnishings show low or negative profit, indicating potential areas for improvement.
- 📅 Sales show a steady upward trend, with the highest sales recorded in the last quarter (October–December).
- 💰 The business generated total sales of ₹196.19M and a total profit of ₹17.58M, achieving an overall profit margin of 8.96%.
- 🛒 A total of 5,010 orders were processed, with an average order value of ₹39K.
- 🎯 The Profit vs Discount analysis suggests that higher discounts are generally associated with lower profitability, especially in the Furniture category.
