# 📊 Sales Dashboard

## Overview
This is an interactive E-Commerce Sales Dashboard built using Power BI. It helps analyze sales performance through interactive visualizations and key performance indicators (KPIs).

## Features
- 📈 Sales Performance Analysis
- 🌍 Region-wise Sales Analysis
- 📅 Monthly Sales Trends
- 📊 Category-wise Sales
- 🎯 Interactive Filters and Slicers

## Tools Used
- Power BI
- Microsoft Excel
